## User Flows

### Sign Up & First Login
```
1. User lands on login page
2. Clicks "Sign in with Google"
3. Google OAuth consent screen
4. User approves permissions
5. Auto-generated username from email
6. Redirected to home page
7. Empty state: "Discover Content" CTA
8. User starts browsing/searching
```

### Add Book to List
```
1. User finds book in search/discover
2. Clicks "Add to List"
3. Selects target list (or creates new)
4. Optional: Adds custom note/date
5. Book added to list
6. Toast confirmation
```

### Write & Publish Review
```
1. User opens book details
2. Clicks "Write Review"
3. Sets star rating (1-5)
4. Writes review text (optional title)
5. Selects reading status
6. Clicks "Publish"
7. Review appears on book page
8. Notifications sent to relevant users
```
